# Minimal Neovim Config

LazyVim-based config with horizon theme and minimal UI.

## Keybinds

| Key | Action |
|-----|--------|
| `<C-p>` | Find files (git only) |
| `<Space>b` | Toggle file tree |
| `<Space>ua` | Toggle autocomplete |
| `<Space>z` | Zen mode |
| `s` | Flash jump |

## Install

```bash
git clone https://github.com/nvidela-dev/neovim-config.git ~/.config/nvim
nvim
```
